# Instrucciones 
1. Crea el repositorio en GitHub
  - Entra en https://github.com y crea una cuenta si no la tienes.
  - Clona este repositorio.
2. Despliega en Streamlit Cloud
  - Ve a https://streamlit.io/cloud y accede con tu usuario de GitHub.
  - Pulsa “New app”.
  - Selecciona tu repositorio, la rama (main) y el archivo principal: app.py.
  - Pulsa “Deploy”.
3. Usa la app
  - Ajusta los sliders de la barra lateral para cambiar el tamaño mínimo de objetos y agujeros.
  - Observa cómo cambian la máscara, el número de regiones y la tabla de características.
